# 🚀 FinControl - Gestão Financeira Inteligente

Bem-vindo ao **FinControl**, um ecossistema completo para controle financeiro pessoal. Este projeto utiliza tecnologias de ponta para oferecer uma experiência segura, rápida e com relatórios de alta fidelidade.

---

## 📂 Estrutura do Repositório

Este repositório está dividido em duas partes principais:

* **[Backend (NestJS)](./fincontrol-backend):** API robusta com autenticação JWT, integração com PostgreSQL via Prisma e lógica de negócios para transações e caixinhas.
* **[Frontend (Next.js)](./fincontrol-frontend):** Interface moderna com Dashboard interativo, sistema de metas estilo Nubank e exportação de PDF premium.

---

## ✨ Funcionalidades em Destaque

- **📦 Caixinhas Multi-objetivos:** Separação de capital para metas específicas (Carro, Viagem, Reserva).
- **📊 Dashboard Dinâmico:** Gráficos vivos e resumos de patrimônio líquido em tempo real.
- **📄 Exportação PDF:** Relatórios profissionais gerados via browser com design minimalista.
- **🛡️ Segurança:** Rotas protegidas e autenticação baseada em tokens.

---

## 🛠️ Tecnologias Principais

**Fullstack TypeScript:**
`Next.js` | `NestJS` | `Tailwind CSS` | `Prisma` | `PostgreSQL` | `Lucide Icons`
